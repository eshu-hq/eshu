# postgresproof

## Purpose

`postgresproof` creates an empty PostgreSQL database for live tests that need
destructive schema or data setup.

## Ownership boundary

This package owns proof-database validation, creation, connection setup, and
cleanup. It does not bootstrap Eshu's schema or seed fixtures. The calling test
owns those steps after it receives the isolated connection.

## Exported surface

`OpenDisposableDatabase` is the only exported function. Its godoc contract is
in [doc.go](doc.go).

## Dependencies

The helper uses `pgx` to parse the administrative DSN and register a copied
connection configuration whose database name is the generated proof database.
It imports no Eshu runtime package.

## Telemetry

This test helper emits no metrics, spans, or structured logs. Test failures
identify the failed lifecycle step without printing the DSN.

## Gotchas / invariants

The package must never accept an application database such as `eshu`, and live
tests must not issue `DROP SCHEMA` against the administrative connection. The
opt-in value must be exactly `1`, and the DSN's database must be exactly
`postgres`. Cleanup may force-drop only a name generated by this package.
